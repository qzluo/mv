源代码各部分说明

1. bin/ 文件夹： 编译生成的可执行文件及运行时依赖的库文件。


2. external/ 文件夹： 依赖的外部库，其中包括：
    -- DaHengImage: 大恒图像库
    -- Vimba: vimba 图像库
    -- OpenCV2.4.13 : opencv 库


3. source/ 文件夹: 跟界面无关的业务相关源代码，主要包括：
    -- algorithm/ 文件夹: 算法相关的代码，包括：
        -- external/文件夹：算法库相关
        -- 其他文件： 算法调用源代码

   -- base/文件夹： 一些基础函数源代码

   -- camera/文件夹： 相机调用接口源代码

   -- comm/文件夹： 串口通信源代码

   -- inspect/文件夹：检测调用接口源代码，包扩图像缓存

   -- spdlog/文件夹： 日志调用源代码


4. zctvision/文件夹： 项目相关的界面及调用上面接口的源代码。
